# 安全策略

## 报告漏洞

如果你发现安全漏洞，请**不要**开公开的 Issue。

请通过 GitHub Security Advisory 报告：

**https://github.com/tianxingleo/dlut-nihongo-quiz/security/advisories/new**

请在报告中包含：

- 漏洞的清晰描述
- 复现步骤
- 影响范围
- 建议的修复方案（如果有）

## 响应时间

- 收到报告后 **48 小时**内确认
- 评估后 **7 天**内给出修复计划

## 适用范围

本策略适用于 `tianxingleo/dlut-nihongo-quiz` 仓库。

题目内容的版权问题不属于安全漏洞，请走 README 中"版权与免责声明"列出的流程。
