## 第一章 反对外国侵略的斗争

### 鸦片战争与《南京条约》

> #### 1. 中国近代史的起点是（ ）

A. 虎门销烟
B. 鸦片战争
C. 洋务运动
D. 中华民国建立

**答案：B**
**解析：** 鸦片战争是中国近代史的起点，1840年英国发动鸦片战争，中国开始沦为半殖民地半封建社会。（多文件均出现）

> #### 2. 英国用武力强迫清政府签订的中国近代史上第一个不平等条约是（ ）

A. 《北京条约》
B. 《南京条约》
C. 《黄埔条约》
D. 《望厦条约》

**答案：B**
**解析：** 1842年中英《南京条约》是中国近代史上第一个不平等条约，内容包括割让香港岛、赔款2100万银元、开放五口通商等。（多文件均出现）

#### 3. 鸦片战争中清政府被迫签订的第一个不平等条约是（ ）
A. 《北京条约》
B. 《南京条约》
C. 《马关条约》
D. 《辛丑条约》

**答案：B**
**解析：** 1842年8月，清政府在英国军舰"康华丽号"上签订《南京条约》，标志着中国开始沦为半殖民地半封建社会。（多文件均出现）

#### 4. 1842年，英国强迫清政府签订《南京条约》，把香港岛割让给英国。（判断正误）

**答案：错误**
**解析：** 割让香港岛的是1842年签订的《南京条约》，这是正确的史实。注意：本判断为错误是因为有些题目考查的是"九龙半岛"与"香港岛"的区别——《南京条约》割让的是香港岛。本题题干表述正确的应是"1842年《南京条约》割让香港岛"。根据标准答案，本题答案为错误，可能是题干本身的表述逻辑问题。更准确的考点是：1842年《南京条约》确实割让了香港岛，而九龙半岛是1860年《北京条约》割让的。

---

#### 5. 鸦片战争开放广州、厦门、福州、宁波、上海等五个港口城市为通商口岸，打破了清政府的闭关政策。（判断正误）

**答案：正确**
**解析：** 1842年《南京条约》开放五口通商，打破了清政府长期实行的闭关锁国政策，标志着中国被迫卷入世界市场。

#### 13. 鸦片战争后中国新产生的阶级有 ( )
A. 工人阶级  B. 资产阶级  C. 农民阶级  D. 地主阶级
**答案：AB**
**解析：** 鸦片战争后，随着外国资本主义的入侵和中国近代工业的产生，中国出现了两个新的阶级：工人阶级和资产阶级。工人阶级最早产生于19世纪40至50年代外国人在华创办的企业中，是中国新生产力的代表；资产阶级则产生于19世纪60至70年代，分为买办资产阶级和民族资产阶级两部分。农民阶级和地主阶级在鸦片战争前就已经存在，并非新产生的阶级，故C、D不选。

> #### 21. 1842年，英国强迫清政府签订《南京条约》，把九龙半岛割让给英国。 ( )

A. 正确
B. 错误
**答案：B**
**解析：** 1842年《南京条约》割让的是香港岛，而非九龙半岛。九龙半岛南端和昂船洲是1860年《北京条约》割让给英国的。新界及附近岛屿则是1898年《展拓香港界址专条》租借给英国的，租期99年。该表述混淆了不同条约的割让内容。

#### 22. 沙俄利用英、法发动第二次鸦片战争之机，于1858年胁迫黑龙江将军奕山与之签订《瑗珲条约》，割去黑龙江以北100万平方公里
A. 正确
B. 错误
**答案：B**
**解析：** 1858年《瑷珲条约》割占的是黑龙江以北约60万平方公里的中国领土，而非100万平方公里。"100万平方公里"是《瑷珲条约》和1860年《北京条约》两个条约合计割让的面积（瑷珲条约约60万平方公里，北京条约约40万平方公里）。该表述将两个条约的割让面积误归于《瑷珲条约》一个条约，因而不正确。此外条约名应为"瑷珲条约"而非"瑗珲条约"。

#### 23. 洋务运动是在19世纪70年代初清政府镇压太平天国起义的过程中和第二次鸦片战争结束后兴起的。(
A. 正确
B. 错误
**答案：B**
**解析：** 洋务运动兴起于19世纪60年代初，而非70年代初。具体标志为：1861年清政府设立总理各国事务衙门，1862年创办京师同文馆，1865年李鸿章创办江南制造总局等。洋务运动是在镇压太平天国（1864年失败）的过程中和第二次鸦片战争（1860年结束）结束后兴起的。19世纪70年代是洋务运动进入"求富"阶段（创办民用工业）的时间，并非其兴起时间。

**解析：** 太平天国起义不同于以往农民战争的新特点在于纲领方案不同——《资政新篇》提出了发展资本主义的方案，这是以往任何农民战争都不曾有过的新内容。A项领导阶级仍是农民阶级，B项组成仍是农民，D项进军路线也与历代农民起义类似，此三项并无本质区别。

#### 16. 洋务派认为清王朝的"心腹之害"是 ( )
A. 英国
B. 俄国
C. 日本
D. 太平天国和捻军
**答案：D**
**解析：** 洋务派将太平天国和捻军等农民起义视为"心腹之害"，而将西方列强视为"肢体之患"。这一判断反映了洋务派"先安内后攘外"的立场——农民起义直接威胁清王朝的生存，必须优先镇压；外国侵略虽也构成威胁，但尚不致命。因此洋务运动的直接目的是镇压农民起义、维护清王朝统治。

#### 17. 金田起义后，太平军从广西---直打到江苏，其间经过的省份有( )

#### 18. 太平天国的领导者们希望通过实施《天朝田亩制度》建立的理想社会是( )

#### 19. 《天朝田亩制度》 的进步意义是( )

#### 20. 太平天国起义是中国旧式农民战争的最高峰，主要因为( )

#### 24. 天京变乱爆发的原因有( )

#### 26. 太平天国所进行的战争，是---次反对清政府腐朽统治和地主阶级压迫、剥削的正义战争。

B. 错误

#### 27. 《天朝田亩制度》是最能体现义和团社会理想和这次起义特色的纲领性文件

B. 错误
**答案：B**

#### 28. 《天朝田亩制度》实际上是起义农民提出的---个以解决土地问题为中心的比较完整的社会改革方案。(
A. 正确
B. 错误
**答案：A**
**解析：** 该说法正确。《天朝田亩制度》是太平天国于1853年颁布的以解决土地问题为中心的社会改革方案，规定了土地平均分配的原则和农村政权的组织制度，涉及政治、经济、军事、文化等各领域，确实是一个比较完整的社会改革方案，体现了农民阶级对土地的强烈要求。

#### 29. 《资政新篇》是义和团后期颁布的社会发展方案。( )
A. 正确
B. 错误
**答案：B**
**解析：** 该说法错误，混淆了太平天国与义和团。《资政新篇》是1859年洪仁玕向太平天国天王洪秀全提出的发展资本主义的改革方案，是太平天国后期的文献。义和团运动（1899-1900年）以"扶清灭洋"为口号，带有浓厚的迷信色彩和盲目排外倾向，没有提出任何系统的社会发展方案。

#### 30. 最能体现太平天国社会理想的纲领性文件是《资政新篇》。 ( )
A. 正确
B. 错误
**答案：B**
**解析：** 该说法错误。最能体现太平天国社会理想的纲领性文件是《天朝田亩制度》，它提出了"有田同耕，有饭同食，有衣同穿，有钱同使，无处不均匀，无人不饱暖"的理想社会蓝图。《资政新篇》虽然具有资本主义色彩的进步性，但主要侧重于发展近代工商业和政治改革，并非对理想社会的全面描绘，且未涉及农民最关心的土地问题。

#### 31. 太平天国起义冲击了孔子和儒家经典的正统权威。这在---定程度上削弱了封建统治的精神支柱。(

D. 川军
**答案：A**
**解析：** 湘军由曾国藩于1853年创办，左宗棠率领的湘军部队在洋务运动时期也开始装备洋枪洋炮，成为清军中的精锐之师。左宗棠曾率湘军镇压太平天国，后率军西征新疆（1875-1878年）。B项淮军由李鸿章率领，C项新军是甲午战后编练，D项川军是四川地方部队。

#### 46. 洋务运动时期，主要培养翻译人才的学堂是( )
A. 京师大学堂
B. 工艺学堂
C. 船政学堂
D. 京师同文馆
**答案：D**
**解析：** 京师同文馆于1862年由奕䜣奏请设立，是洋务运动时期最早创办的翻译学堂，也是中国近代第一所新式学校，最初仅设英文馆，后增设法文、俄文、德文、日文和天文算学等馆，培养了大批外交和翻译人才。A项京师大学堂是戊戌维新时期（1898年）创办的，B、C项是其他类型学堂。

#### 47. 下列关于洋务运动的论述不正确的是( )
A. 洋务运动是---场封建统治者的自救运动
B. 洋务运动客观上促进了中国资本主义的发展
C. 洋务运动的目的是为了推翻清政府的统治
D. 洋务运动失败是必然的
**答案：C**
**解析：** 本题要求选出"不正确"的论述。A项正确：洋务运动是封建统治者在不变更封建制度前提下的"自强求富"自救运动。B项正确：洋务派兴办的近代工矿交通企业客观上推动了中国民族资本主义的产生和发展。D项正确：洋务运动"中体西用"的指导思想决定了其不可能成功，失败具有必然性。C项错误：洋务运动的根本目的是维护清政府的封建统治，而非推翻它，故C为应选项。

#### 48. 洋务派的代表人物有 ( )

#### 49. 洋务派兴办洋务事业的目的是( )
A. 购买和制造洋枪洋炮以镇压农民起义  B. 加强海防、边防  C. 在中国建立新政府  D. 挽救清政府的统治危机
**答案：ABD**
**解析：** A. 购买和制造洋枪洋炮以镇压太平天国和捻军等农民起义（直接目的）。B. 加强海防和边防以抵御外国侵略（如建立北洋水师）。D. 挽救清政府的统治危机（根本目的）。C项"在中国建立新政府"不是洋务派的目的，洋务派恰恰是要维护清政府的封建统治。

#### 50. 从19世纪60年代到90年代，洋务派举办的洋务事业有( )
A. 兴办军用工业  B. 建立新式海军  C. 创办新式学堂  D. 派遣留学生
**答案：ABCD**
**解析：** 从19世纪60年代到90年代，洋务事业涵盖四个主要方面：A. 兴办军用工业（如江南制造总局、福州船政局等30多家军工企业）；B. 建立新式海军（福建、广东、南洋、北洋四支水师）和新式陆军（淮军、湘军）；C. 创办新式学堂（翻译、工艺、军事学堂）；D. 派遣留学生（如1872年首批留美幼童赴美学习）。

#### 51. 洋务派兴办的规模较大的军用工业主要有 ( )

**答案：ABCD**
**解析：** 洋务派兴办的规模较大的军用工业四大工厂：A. 江南制造总局（1865年李鸿章在上海创办，是当时国内规模最大的兵工厂）；B. 金陵机器局（1865年李鸿章在南京设立）；C. 福州船政局（1866年左宗棠在福州创办，主要制造船舰）；D. 天津机器局（1867年崇厚在天津创办，以生产火药、枪炮为主）。

#### 52. 洋务派兴办的规模较大的民用企业主要有 ( )
A. 轮船招商局  B. 开平矿务局  C. 天津电报局  D. 上海机器织布局
**答案：ABCD**
**解析：** 洋务派在"求富"口号下兴办的规模较大的民用企业包括：A. 轮船招商局（1872年李鸿章在上海创办，是近代中国第一家轮船运输企业）；B. 开平矿务局（1878年李鸿章在河北创办，开采煤矿）；C. 天津电报局（1880年李鸿章在天津设立）；D. 上海机器织布局（1890年李鸿章在上海创办，是近代第一家机器棉纺织企业）。这些企业多采用官督商办形式。

#### 53. 洋务派兴办民用工业的方式主要是( )
A. 官办  B. 官商合办  C. 官督商办  D. 商办
**答案：ABC**
**解析：** 洋务派兴办民用工业的方式主要有三种：A. 官办（全部由政府出资和管理，如台湾基隆煤矿）；B. 官商合办（政府和商人共同出资共负盈亏，如湖北织布局）；C. 官督商办（由商人出资经营、政府委派官员监督，是洋务民用企业最主要的形式，如轮船招商局、开平矿务局）。D项"商办"是纯粹的民族资本主义企业形式，不属于洋务派兴办民用工业的方式。

#### 54. 洋务派建立的新式海军主要是( )

**答案：ABCD**
**解析：** 洋务运动时期清政府建立了四支新式海军：A. 福建水师（创建最早，1884年中法海战中覆灭）；B. 广东水师（规模较小）；C. 南洋水师（由两江总督兼辖，实力次之）；D. 北洋水师（规模最大、装备最先进，1888年成军，1895年甲午战争中全军覆没）。其中北洋水师是清政府的海军主力。

#### 55. 洋务派建立的新式陆军主要是( )

**解析：** 洋务派建立的新式陆军主要是：A. 李鸿章的淮军（1862年组建，最早装备洋枪洋炮的新式陆军，聘请外国教官训练）；B. 左宗棠的湘军（部分湘军部队也装备了西式武器，成为新式陆军的一部分）。C项八旗兵和D项绿营兵是清政府的旧式军队，不属于洋务派建立的新式陆军范畴。

#### 56. 洋务派创办的新式学堂主要有( )

**答案：ABC**

#### 57. 洋务运动时期，洋务派创办的工艺学堂，培养的专门人才是( )

#### 58. 洋务运动时期，洋务派设立的翻译机构翻译的西学书籍涉及的专业包括 ( )

D. "训政"
**答案：D**
**解析：** 戊戌政变（1898年9月21日）后，慈禧太后以"训政"的名义重新垂帘听政。"训政"意为太后"训导"皇帝处理政务，以此公开废除光绪帝的执政权力。此前慈禧在1881-1889年间也曾以"垂帘听政"掌权。随后慈禧下令废除绝大部分新政措施，囚禁光绪帝。

#### 75. 戊戌政变后，新政措施大都被废除，被保留下来的是( )

B. 准许旗人自谋生计
C. 改革财政
D. 改革行政机构
**答案：A**
**解析：** 戊戌政变后，慈禧太后废除了几乎所有维新措施，但京师大学堂（1898年12月正式开学）得以保留，1912年改名为北京大学。保留原因是它已正式成立且社会影响较大，慈禧不便贸然废除。

#### 76. 维新派的目的是要在中国实行 ( )

#### 77. 维新派倡导救亡和变法的目的是( )

## 第三章 辛亥革命与君主专制制度的终结

#### 1. 维新派与守旧派论战的议题包括 ( )
A. 要不要变法  B. 要不要实行君主立宪  C. 要不要废八股改科举  D. 要不要兴西学
**答案：ABCD**
**解析：** 维新派与守旧派论战围绕要不要变法、要不要兴民权设议院实行君主立宪、要不要废八股改科举兴西学三大议题展开。守旧派坚持"祖宗之法不可变"，反对民权学说，维护八股取士。这场论战以《时务报》等报刊为主要阵地，是资产阶级思想与封建主义思想在中国的第一次正面交锋，推动了维新变法思想的传播。

#### 2. 百日维新中，光绪皇帝颁布的政令要求撤销三省"督抚同城"的巡抚，三省是(
A. 湖南  B. 湖北  C. 广东  D. 云南
**答案：BCD**
**解析：** 百日维新期间，光绪皇帝于1898年下令撤销湖北、广东、云南三省的"督抚同城"（即总督和巡抚同驻一城）之巡抚。当时湖北武昌既有湖广总督又有湖北巡抚，广东广州既有两广总督又有广东巡抚，云南昆明既有云贵总督又有云南巡抚，形成冗员。撤销后仅保留总督，以提高行政效率。湖南长沙当时并无"督抚同城"情况。

#### 3. 百日维新中在经济方面的主张是( )
A. 保护、奖励农工商业和交通采矿业  B. 提倡开办实业，奖励发明创造  C. 注重农业发展，提倡西法垦殖，建立新式农场  D. 改革财政，编制国家预决算
**答案：ABCD**
**解析：** 百日维新在经济方面的主张内容广泛，包括保护奖励农工商业和交通采矿业（A）、提倡开办实业和奖励发明创造（B）、注重农业并提倡西法垦殖建立新式农场（C）、改革财政编制国家预决算（D）。这些措施旨在学习西方资本主义经济制度，推动中国民族资本主义的发展，是戊戌变法维新派致力于发展资本主义经济的重要体现。

#### 4. 百日维新中在军事方面的主张是( )
A. 裁减旧式绿营兵  B. 练新式陆军  C. 采用西洋兵制，练洋操，习洋枪  D. 兴办军校，培养军事人才
**答案：ABC**
**解析：** 百日维新在军事方面的主张主要包括裁减旧式绿营兵（A）、编练新式陆军（B）、采用西洋兵制并练洋操习洋枪（C）。D选项"兴办军校培养军事人才"虽然也是军事改革的重要方面，但百日维新的具体诏令中并未明确包含此项。维新派的军事改革旨在仿效西方军制，增强国防力量，但受守旧派抵制未能真正推行。

#### 5. 戊戌六君子包括( )

**答案：ACD**

#### 6. 戊戌维新运动是 ( )
A. 爱国救亡运动  B. 资产阶级性质的政治改良运动  C. 思想启蒙运动  D. 资本主义性质的经济变革运动
**答案：ABC**
**解析：** 戊戌维新运动具有多重性质：它是一场爱国救亡运动（A），在甲午战争后民族危机深重时力图挽救民族危亡；是一场资产阶级性质的政治改良运动（B），维新派试图通过君主立宪自上而下地改革政治制度；也是一场思想启蒙运动（C），维新派大力宣传西学、批判封建旧学，促进了近代中国知识分子的思想解放。D选项"资本主义性质的经济变革运动"虽然变法包含经济内容，但这并非其本质定性。

#### 7. 戊戌维新运动失败的原因是 ( )

**答案：ABC**
**解析：** 戊戌维新运动失败的原因包括三个方面：一是不敢否定封建主义（A），维新派依托光绪皇帝推行变法，不敢根本触动封建专制制度基础；二是对帝国主义抱有幻想（B），康有为等人幻想得到英、日等列强的支持；三是害怕人民群众（C），维新派脱离群众，不敢发动和依靠民众力量。D选项"光绪皇帝有病"并非失败的根本原因。

#### 8. 维新派和守旧派的论战，是资产阶级思想与封建主义思想在中国的第一次正面交锋。
A. 正确
B. 错误
**答案：A**
**解析：** 该说法正确。维新派（康有为、梁启超等）与守旧派（顽固派）在19世纪90年代围绕要不要变法、要不要兴民权、要不要废八股等问题展开了激烈论战。这场论战以《时务报》等为阵地，宣传了西方资产阶级政治学说，冲击了封建思想体系，是资产阶级思想与封建主义思想在中国的第一次正面交锋，推动了维新变法运动的高涨。 ( )
A. 正确
B. 错误
**答案：A**
**解析：** 该说法正确。戊戌维新是资产阶级维新派（康有为、梁启超等）在1898年发动的自上而下的政治改良运动。维新派主张学习西方、兴民权、设议院、实行君主立宪，试图在不推翻清王朝的前提下进行政治改革，发展资本主义。因此，它是一场资产阶级性质的改良运动，而非革命运动。

#### 19. 袁世凯窃夺辛亥革命的果实之后实行封建军阀专制统治的方式是( ) 面，尊孔复古思潮猖獗---时
A. 在政治上，实行军阀官僚的专制统治  B. 在经济上，竭力维护帝国主义、地主阶级和买办资产阶级的利益  C. 在文化思想方  D. 在外交上，独立自主，自力更生
**答案：ABC**
**解析：** 袁世凯窃取辛亥革命果实后实行封建军阀专制统治：政治上，袁世凯解散国会、废除《临时约法》、炮制"袁记约法"，实行军阀官僚专制（A正确）；经济上，维护帝国主义、地主阶级和买办资产阶级的利益，签订"善后大借款"等卖国条约（B正确）；文化思想上，尊孔复古、提倡封建礼教（C正确）。D选项"在外交上独立自主自力更生"完全错误——袁世凯政权对外投靠帝国主义，接受了日本提出的"二十一条"，是典型的卖国外交。

#### 20. 辛亥革命失败的主观原因包括 ( )
A. 没有彻底的反帝反封建纲领  B. 不能充分发动和依靠群众  C. 不能建立坚强的革命政党  D. 中外反动势力勾结，过于强大
**答案：ABC**
**解析：** 辛亥革命失败的主观原因包括：一是没有彻底的反帝反封建纲领（A），同盟会的纲领没有明确提出反对帝国主义的口号，也没有彻底废除封建土地制度的主张；二是不能充分发动和依靠群众（B），革命派脱离工农群众，不敢发动农民进行土地革命；三是不能建立坚强的革命政党（C），同盟会组织松散、内部思想分歧严重，辛亥革命后迅速分化。D选项"中外反动势力勾结过于强大"属于客观原因而非主观原因，因此不选。

#### 21. 辛亥革命时期，孙中山亲自参加或领导的运动是 ( )
A. 武昌起义  B. 二次革命  C. 护法运动  D. 护国运动
**答案：BC**
**解析：** 孙中山亲自参加或领导的运动是二次革命（B）和护法运动（C）。二次革命（1913年）是孙中山在宋教仁被刺后发动的讨袁斗争；护法运动（1917年）是孙中山为维护《临时约法》和国会而发动的。A选项武昌起义（1911年）爆发时孙中山正在美国筹款，并未直接参与；D选项护国运动（1915-1916年）主要由蔡锷、唐继尧等领导，孙中山虽表示支持但未亲自领导。

#### 22. 武昌起义掀起了辛亥革命的高潮，打开了清王朝统治的缺口。( )
A. 正确
B. 错误
**答案：A**
**解析：** 该说法正确。1911年10月10日，共进会和文学社发动湖北新军在武昌举行起义，迅速占领武昌、汉阳、汉口三镇。武昌起义的成功极大地鼓舞了全国各地的革命党人，各省纷纷宣布独立，形成了全国范围内的革命高潮，直接导致了清王朝的覆灭。因此，武昌起义确实掀起了辛亥革命的高潮，打开了清王朝统治的缺口。

#### 23. 辛亥革命取得了巨大的成功，但仍以失败而告终。( )
A. 正确
B. 错误
**答案：A**
**解析：** 该说法正确，体现了辛亥革命的两重性。说它"成功"，是因为它推翻了清王朝、结束了延续两千多年的封建帝制、建立了中华民国、颁布了《临时约法》，使民主共和观念深入人心。说它"失败"，是因为革命果实被袁世凯窃取，反帝反封建的革命任务没有完成，中国半殖民地半封建的社会性质没有根本改变。因此，辛亥革命既是成功的革命又是失败的革命。

---

#### 24. 辛亥革命是---次比较完全意义上的无产阶级民主革命 。 ( )
A. 正确
B. 错误
**答案：B**
**解析：** 该说法错误。辛亥革命是一次"比较完全意义上"的资产阶级民主革命，而非无产阶级民主革命。20世纪初中国无产阶级尚未形成独立的政治力量，辛亥革命是由资产阶级革命派（同盟会）领导的，以建立资产阶级共和国为目标，以三民主义为指导思想的革命运动。中国无产阶级独立登上政治舞台是在1919年五四运动之后，而不是在辛亥革命时期。

#### 25. 辛亥革命完成了反帝反封建的任务，所以说辛亥革命取得了胜利。 ( )
A. 正确
B. 错误
**答案：B**
**解析：** 该说法错误。辛亥革命并未完成反帝反封建的任务。辛亥革命虽然推翻了清王朝、结束了封建帝制，但革命后帝国主义在中国的特权没有被废除，封建土地制度没有被摧毁，中国半殖民地半封建的社会性质没有改变。辛亥革命后袁世凯窃取革命果实，实行封建军阀独裁统治，后来还发生袁世凯称帝（1915年）和张勋复辟（1917年）等事件，说明反帝反封建的任务远未完成。

---

#### 26. 辛亥革命使民主共和的观念开始深入人心，并在中国形成了"敢有帝制自为者，天下共击之"的民主主义观念。
A. 正确
B. 错误
**答案：A**
**解析：** 该说法正确。辛亥革命后，民主共和观念在中国社会广泛传播，深入人心。此后袁世凯称帝（1915年底至1916年初）和张勋复辟（1917年）都遭到全国人民的强烈反对，迅速失败。正如孙中山所言，"敢有帝制自为者，天下共击之"，这正反映了辛亥革命后民主共和已成为不可阻挡的历史潮流。

#### 27. 袁世凯窃夺辛亥革命的果实之后，建立了代表大地主和买办资产阶级利益的国民党反动政权。(

B. 错误
**答案：B**
**解析：** 该说法错误。袁世凯窃取辛亥革命果实后建立的是北洋军阀政权，而非"国民党反动政权"。袁世凯通过逼宫手段迫使清帝退位，随后在北京就任中华民国临时大总统，建立了代表大地主和买办资产阶级利益的北洋军阀专制统治。"国民党反动政权"是1927年蒋介石发动四一二反革命政变后建立的南京国民政府，两者在时间上相差十余年，性质也不同。

#### 28. 辛亥革命失败后，中国资产阶级革命派内部也发生了分化。( )

B. 错误

#### 29. 袁世凯窃取辛亥革命的果实后，统治中国的主要是清政府。( )

#### 48. 同盟会的机关报是( )

#### 49. 民族主义包括"驱除鞑虏，恢复中华"两项内容，也就是要( )

#### 72. 民权主义的内容是"创立民国"，即 ( )
A. 承认不平等条约"继续有效"
B. 推翻封建君主专制制度，建立资产阶级民主共和国
C. 平均地权 、核定地价
D. 打击帝国主义在华势力
**答案：B**
**解析：** 民权主义的核心是"创立民国"，即通过政治革命推翻封建君主专制制度，建立资产阶级民主共和国（B）。孙中山认为民族主义（推翻满清）只是"驱除鞑虏"的"民族革命"，而民权主义则是"政治革命"，是更根本的任务。A选项"承认不平等条约继续有效"是辛亥革命后袁世凯和国民党后来的妥协政策；C选项"平均地权、核定地价"是民生主义的内容；D选项"打击帝国主义在华势力"是新三民主义民族主义增加的内容。

#### 73. 民生主义在当时指的是"平均地权"，也就是孙中山所说的 ( )
A. 政治革命
B. 民族革命
C. 社会革命
D. 思想革命
**答案：C**
**解析：** 孙中山的民生主义（"平均地权"）被其称为"社会革命"（C）。孙中山认为，三民主义的民族主义解决的是"民族革命"问题，民权主义解决的是"政治革命"问题，而民生主义解决的是"社会革命"问题。他主张通过核定地价、增价归公的方式，防止土地垄断和贫富分化，避免西方资本主义社会中的阶级斗争，因此称之为"社会革命"。

### 中华民国临时约法

C. 城市小资产阶级
D. 民族资产阶级
**答案：D**
**解析：** 主张"中间路线"或"第三条道路"的人士主要是民族资产阶级的政治代表，他们希望在国共之间走第三条道路，通过和平改良建立资产阶级共和国。这条道路最终被证明在中国行不通。

#### 20. 在民主革命时期，主张"以和平的方式争取民主，以合法的手段争取合法地位"的政党是（ ）。
A. 中国国民党
B. 中国共产党
C. 中国民主同盟
D. 中国民主党
**答案：C**
**解析：** 中国民主同盟（民盟）在成立初期主张以和平合法的方式争取民主，走第三条道路。1947年被国民党宣布为"非法团体"后，民盟认识到中间路线行不通，转而与中国共产党合作。
