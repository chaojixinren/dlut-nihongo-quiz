## 五四运动（新民主主义革命开端）

#### 1. [来源：1-大工题库（分章节）.md]

在1919年五四运动至1949年新中国成立以前这个时期，中国仍然是半殖民地半封建社会。(
)

A. 正确

B. 错误

**答案：A**

**解析：** 1840年鸦片战争后中国逐步沦为半殖民地半封建社会，1919年五四运动时中国仍处于北洋军阀统治之下，帝国主义和封建主义的压迫并未消除，直到1949年新中国成立才结束这一社会形态。因此该说法符合历史事实。

#### 2. [来源：1-大工题库（分章节）.md]

五四运动的直接导火线是 ( )

A. 袁世凯接受日本提出的"⼆⼗---条"

B. 俄国十月革命的胜利

C. 巴黎和会上中国外交的失败

D. 李大钊《庶民的胜利》的发表

**答案：C**

**解析：** 五四运动的直接导火线是1919年巴黎和会上中国外交的失败。

#### 3. [来源：1-大工题库（分章节）.md]

五四运动发生的时代条件是( )

A. 新的社会力量的成长壮大  B. 新文化运动掀起的思想解放潮流  C. 日本强割台湾岛  D. 俄国十月革命的影响
**答案：ABD**

**解析：** 五四运动爆发的时代条件包括：一战期间民族资本主义发展推动工人阶级和资产阶级等新力量成长壮大（A）；新文化运动（1915年起）掀起思想解放潮流，为五四运动准备了群众和骨干（B）；1917年俄国十月革命给中国送来了马克思主义，推动了先进分子向俄国学习（D）。C项"日本强割台湾岛"是1895年《马关条约》内容，不是五四运动的时代条件。

#### 4. [来源：1-大工题库（分章节）.md]

五四运动中迫于人民群众的压力，
被北洋政府不得不罢免了职务的三个卖国贼是( )

A. 梁士诒  B. 曹汝霖  C. 章宗祥  D. 陆宗舆
**答案：BCD**

**解析：** 五四运动中，广大学生和民众强烈要求罢免亲日派卖国贼。曹汝霖（交通总长，参与"二十一条"谈判）、章宗祥（驻日公使，出卖山东权益）、陆宗舆（币制局总裁，参与"二十一条"借款）三人因出卖国家权益遭到全国声讨。北洋政府迫于压力于1919年6月10日下令罢免三人。梁士诒虽也是北洋要员，但不在此次罢免之列。

#### 5. [来源：1-大工题库（分章节）.md]

1919年五四运动以前的新文化运动是资产阶级民主主义的新文化反对封建主义的旧文化的斗争。(
)

A. 正确

B. 错误

**答案：A**

**解析：** 五四前的新文化运动（1915-1919年）以陈独秀创办《新青年》为标志，提倡民主与科学，反对封建专制和迷信盲从，其指导思想是资产阶级民主主义，而非马克思主义。李大钊发表《青春》《庶民的胜利》等文章宣传民主主义思想，尚未转向马克思主义。因此该说法符合历史事实。

#### 6. [来源：1-大工题库（分章节）.md]

1919年5月爆发的五四运动，是中国近代史上的---个划时代的事件。( )

A. 正确

B. 错误

**答案：A**

**解析：** 五四运动是中国近代史上一个划时代的事件，因为运动中中国工人阶级以独立姿态登上政治舞台，标志着新民主主义革命的开端。此后中国革命领导阶级由资产阶级转变为无产阶级，革命前途由建立资产阶级共和国转变为建立人民民主专政的国家。

#### 7. [来源：1-大工题库（分章节）.md]

五四运动表现了反帝反封建的不彻底性。( )

A. 正确

B. 错误

**答案：B**

**解析：** 五四运动是一次彻底的反帝反封建的爱国运动。运动中明确提出"外争主权，内除国贼"口号，坚决反对帝国主义和封建军阀。与以往改良运动和革命运动不同，五四运动表现出前所未有的坚决性。说其"不彻底"是混淆了五四运动与旧民主主义革命的区别。

#### 8. [来源：1-大工题库（分章节）.md]

五四运动以前，社会主义思潮在中国蓬勃兴起，马克思主义开始在知识界中得到传播。(
)

A. 正确

B. 错误

**答案：B**

**解析：** 五四运动前虽有零星介绍社会主义的文章，但社会主义思潮在中国蓬勃兴起、马克思主义在知识界广泛传播是在五四运动之后。1919年五四运动后，李大钊发表《我的马克思主义观》系统介绍马克思主义，各地出现研究马克思主义的团体，马克思主义才真正在中国广泛传播。

#### 18. [来源：3-四套机考题.md]

五四运动以前的新文化运动是马克思主义的新文化反对封建主义的旧文化的斗争。F

**答案：F**
**解析：** 五四运动以前的新文化运动（1915-1919年）是资产阶级民主主义的新文化反对封建主义旧文化的斗争，其指导思想是资产阶级民主主义，而非马克思主义。马克思主义在中国的广泛传播是在五四运动之后。

#### 19. [来源：3-四套机考题.md]

1919年爆发的五四运动是（C）

A. 中国社会主义革命阶段的开端

B. 世界社会主义革命阶段的开端

C. 中国新民主主义革命阶段的开端

D. 中国旧民主主义革命阶段的开端

**答案：C**
**解析：** 五四运动（1919年5月4日）是中国近代史上一次彻底的反帝反封建的爱国运动，标志着中国新民主主义革命的开端。新民主主义革命与旧民主主义革命的根本区别在于领导阶级不同——前者由无产阶级领导，后者由资产阶级领导。

#### 20. [来源：3-四套机考题.md]

1919年5月爆发的五四运动，是在新的时代条件和社会历史条件下发生的（ABC）

A. 俄国十月革命对中国产生重要影响

B. 新文化运动掀起的思想解放潮流为五四运动准备了群众队伍和骨干力量

C. 中国资产阶级和工人阶级等新的社会力量成长壮大

D. 马克思主义在中国广泛传播

**答案：ABC**
**解析：** 五四运动的时代条件包括：俄国十月革命的影响（A）、新文化运动的思想解放作用（B）、新的社会力量（资产阶级和工人阶级）的成长壮大（C）。D选项"马克思主义在中国广泛传播"是五四运动之后才出现的情况，而非五四运动发生的条件。

#### 21. [来源：3-四套机考题.md]

1919年5月4日，五四运动爆发。中国新民主主义革命由此开端。T

**答案：T**
**解析：** 1919年5月4日，北京学生率先举行示威游行，反对巴黎和会将德国在山东的权益转让给日本，五四运动由此爆发。五四运动是以无产阶级（工人阶级）为领导力量、以马克思主义为思想指导的彻底反帝反封建运动，标志着中国新民主主义革命的开端。

#### 22. [来源：3-四套机考题.md]

五四运动为中国共产党的成立作了思想上和干部上的准备。T

**答案：T**
**解析：** 五四运动促进了马克思主义在中国的广泛传播（思想准备），并在运动中锻炼和培养了一批先进知识分子和骨干力量（干部准备）。这些经过五四运动洗礼的先进分子——如陈独秀、李大钊、毛泽东等——后来成为中国共产党的重要创始人。

#### 23. [来源：4-近代史习题集.md]

在五四运动至新中国成立前这一时期，中国的社会性质是（ ）。

A. 封建社会

B. 半殖民地社会

C. 资本主义社会

D. 半殖民地半封建社会

**答案：D**
**解析：** 从1919年五四运动到1949年新中国成立，中国的社会性质始终是半殖民地半封建社会。虽然经历了北洋政府和国民党政府两个时期，但中国的社会性质没有发生根本改变。

#### 24. [来源：4-近代史习题集.md]

从五四运动到新中国成立前，中国反帝反封建的革命性质属于（ ）。

B. 农民阶级革命

C. 资产阶级民主主义革命

D. 小资产阶级革命

**答案：C**
**解析：** 五四运动后，虽然工人阶级成为领导力量，但革命任务仍然是反帝反封建，因此革命性质仍然是资产阶级民主主义革命，属于新民主主义革命范畴。

#### 25. [来源：4-近代史习题集.md]

A. 工人阶级

B. 农民阶级

C. 城市小资产阶级

D. 民族资产阶级

**答案：A**
**解析：** 五四运动后，中国工人阶级（通过中国共产党）取代资产阶级，成为中国民主革命的领导力量。这是新旧民主主义革命的根本区别，标志着中国革命进入新民主主义革命阶段。

#### 26. [来源：4-近代史习题集.md]

在五四运动到新中国成立前这一时期，中国反帝反封建的主力是（ ）。

B. 农民阶级

C. 城市小资产阶级

D. 民族资产阶级

**解析：** 农民阶级是民主革命的主力军。中国农民占人口绝大多数，深受帝国主义和封建主义的剥削压迫，具有强烈的革命要求。中国共产党通过土地革命广泛发动农民，建成了巩固的工农联盟。

#### 27. [来源：4-近代史习题集.md]

#### 28. [来源：4-近代史习题集.md]

#### 29. [来源：4-近代史习题集.md]

五四运动的直接导火线，是巴黎和会上中国外交的失败。

**答案：正确**
**解析：** 1919年巴黎和会上，中国作为战胜国提出的取消"二十一条"、收回山东主权等正当要求被列强拒绝，反而决定将德国在山东的权益转让给日本。这一消息传回国内，直接引发了五四爱国运动。

#### 30. [来源：4-近代史习题集.md]

**答案：错误**
**解析：** 五四运动后，新文化运动并未中断反封建的思想启蒙工作，而是进一步发展了。马克思主义的传播不仅没有削弱反封建的任务，反而为反封建斗争提供了更科学的思想武器。

#### 31. [来源：4-近代史习题集.md]

五四运动的直接导火线是（ ）。

A. 俄国十月革命的影响

B. 巴黎和会上中国外交的失败

C. 新文化运动的推动

D. 中国共产党的成立

**答案：B**
**解析：** 1919年巴黎和会中国外交失败（列强将德国在山东的权益转让给日本）是五四运动的直接导火线。消息传到国内，学生率先罢课游行，随后工人罢工、商人罢市，形成了全国范围的爱国运动。

#### 32. [来源：4-近代史习题集.md]

五四运动后期的斗争的主力是（ ）。

A. 学生

B. 工人

C. 商人

D. 农民

**答案：B**
**解析：** 五四运动后期（6月3日后），斗争的中心从北京转移到上海，主力从学生转变为工人。上海等地工人举行政治大罢工，标志着中国工人阶级以独立的姿态登上政治舞台。

#### 33. [来源：4-近代史习题集.md]

**解析：** 五四运动后期（6月3日后），由于学生大批被捕，斗争中心从北京转移到上海。上海工人率先举行政治罢工，随后全国各地的工人、商人也纷纷响应，形成了全国范围的群众性爱国运动。

#### 34. [来源：4-近代史习题集.md]

B. 新的社会力量的成长、壮大

**解析：** 八七会议于1927年8月7日在湖北汉口秘密召开。会议纠正了陈独秀的右倾机会主义错误，确定了实行土地革命和武装反抗国民党反动派的总方针，毛泽东在会上提出了"枪杆子里出政权"的重要论断。

#### 33. [来源：3-四套机考题.md]

新文化运动是中国新民主主义革命阶段的开端。F

**答案：F**
**解析：** 新文化运动（1915年）是资产阶级民主主义的思想文化运动，不属于新民主主义革命范畴。中国新民主主义革命阶段的开端是1919年的五四运动。五四运动以无产阶级为领导，以马克思主义为指导思想，是新民主主义革命的开端。

#### 34. [来源：3-四套机考题.md]

新文化运动兴起的标志是（B）

A. 鲁迅发表《狂人日记》

B. 陈独秀在上海创办《青年杂志》

C. 胡适发表《文学改良刍议》

D. 蔡元培就任北京大学校长

**答案：B**
**解析：** 1915年9月，陈独秀在上海创办《青年杂志》（第二卷起改名《新青年》），新文化运动由此兴起。《新青年》以民主和科学为旗帜，向封建专制和封建礼教发起了猛烈冲击。鲁迅的《狂人日记》（1918年）是新文学的代表作，胡适的《文学改良刍议》（1917年）倡导白话文。

#### 35. [来源：3-四套机考题.md]

1915年，《青年杂志》（后改称《新青年》）创刊，新文化运动开始。T

**答案：T**
**解析：** 1915年9月，陈独秀在上海创办《青年杂志》（1916年第二卷起改名《新青年》），标志着新文化运动的兴起。《新青年》以民主和科学为基本口号，向封建专制和封建礼教发起了猛烈冲击，成为新文化运动的主要阵地。

#### 36. [来源：4-近代史习题集.md]

新文化运动的目的是推动一场新的政治革命。

**答案：错误**
**解析：** 新文化运动（1915年开始）是一场思想文化运动，而非政治革命运动。其目的是通过批判封建思想文化、提倡民主与科学，实现国民性的改造和思想的解放，为政治变革创造思想条件。

#### 37. [来源：4-近代史习题集.md]

新文化运动的基本口号是"打倒孔家店"。

**答案：错误**
**解析：** 新文化运动的基本口号（旗帜）是"民主"与"科学"（即"德先生"和"赛先生"），而非"打倒孔家店"。"打倒孔家店"是新文化运动中激进派批判儒家思想的口号，但不能概括新文化运动的基本口号。

#### 38. [来源：4-近代史习题集.md]

新文化运动既是一场思想启蒙运动，也是一场思想解放运动。

**答案：正确**
**解析：** 新文化运动通过提倡民主与科学、反对封建礼教和迷信，极大地解放了人们的思想，具有思想启蒙的性质。它是近代中国第一次大规模的思想解放运动，为马克思主义在中国的传播创造了条件。

#### 39. [来源：4-近代史习题集.md]

新文化运动中陈独秀对民主的理解就是资产阶级民主主义的制度。

**答案：错误**
**解析：** 五四运动后，陈独秀逐渐转向马克思主义，对民主的理解超越了资产阶级民主主义的范畴，开始主张无产阶级民主和社会主义民主。题干说"就是"过于绝对。

#### 40. [来源：4-近代史习题集.md]

五四前的新文化运动存在形式主义看问题的倾向。

**答案：正确**
**解析：** 五四前的新文化运动存在形式主义的问题，即对传统文化全盘否定、对西方文化全盘肯定的倾向。这种形而上学的方法论不利于科学地分析问题，是早期新文化运动的局限之一。

#### 41. [来源：4-近代史习题集.md]

在五四前，新文化运动的领导人物对资产阶级民主主义是完全相信的。

**答案：错误**
**解析：** 五四前，新文化运动的领导人物普遍对资产阶级民主主义抱有希望，但在实践中屡遭挫折。一战的爆发暴露了资本主义制度的弊端，使一些先进分子开始对资产阶级民主主义产生怀疑。

#### 42. [来源：4-近代史习题集.md]

五四运动后的新文化运动中断了五四以前的反封建的思想启蒙工作。

**答案：错误**
**解析：** 五四运动后，新文化运动并未中断反封建的思想启蒙工作，而是进一步发展了。马克思主义的传播不仅没有削弱反封建的任务，反而为反封建斗争提供了更科学的思想武器。

## 民族资产阶级两面性

#### 1. [来源：2-纲要分章题库.md]

中国民族资产阶级对外国资本主义和本国封建主义（ ）

A、是完全对立，坚决反对的 B、是完全依赖，坚决追随的

C、既有矛盾,斗争的一面，又有依赖,妥协的一面

D、只反对本国封建主义，不反对外国资本主义

## 护法运动/旧民主主义终结

#### 1. [来源：1-大工题库（分章节）.md]

袁世凯在北京就任中华民国临时大总统标志着旧民主主义革命的终结 。 ( )

A. 正确

B. 错误

**答案：B**

**解析：** 旧民主主义革命终结的标志是护法运动（1917-1918年）的失败，而非袁世凯就任临时大总统（1912年3月）。袁世凯就职标志着辛亥革命成果被窃取，但此后孙中山还领导了二次革命（1913年）、护国运动（1915年）、护法运动（1917年）等斗争。护法运动失败才标志旧民主主义革命的终结。

#### 2. [来源：4-近代史习题集.md]

孙中山领导护法运动，所护的"法"是《中华民国约法》。

**答案：错误**
**解析：** 护法运动所护的是《中华民国临时约法》（1912年颁布），而非《中华民国约法》（袁世凯1914年颁布的"袁记约法"）。段祺瑞拒绝恢复临时约法和国会，孙中山遂发动护法运动。

#### 3. [来源：4-近代史习题集.md]

**答案：正确**
**解析：** 1917-1918年的护法运动失败后，孙中山认识到依靠军阀不可能实现民主共和。护法运动的失败证明中国资产阶级领导的旧民主主义革命已经走到尽头，旧民主主义革命终结了。

C、甲午中日战争 D、八国联军侵华战争


**答案：C**
**解析：** 诗中"去年今日割台湾"指甲午战争后清政府被迫将台湾割让给日本，丘逢甲表达了痛失故土的悲愤之情。

#### 6. [来源：3-四套机考题.md]

清政府于1901年4月成立督办政务处，宣布实行______。

**答案："新政"**
**解析：** 1901年，清政府为应对义和团运动和八国联军侵华后的统治危机，宣布实行"新政"。主要内容包括编练新军、改革官制、废除科举、兴办新式学堂等，但并未触动封建专制制度，未能挽救清王朝的覆亡命运。

#### 7. [来源：3-四套机考题.md]

#### 2. [来源：1-大工题库（分章节）.md]

资本-帝国主义对中国的侵略包括( )

A. 政治控制  B. 文化渗透  C. 军事侵略  D. 经济掠夺
**答案：ABCD**

**解析：** 资本-帝国主义对中国的侵略是全方位、多层次的：军事侵略包括发动鸦片战争（1840）、甲午战争（1894）等多次侵华战争；政治控制包括控制中国海关、划定租界、享有领事裁判权等；经济掠夺包括勒索赔款（《辛丑条约》赔款4.5亿两）、控制中国交通金融等；文化渗透包括传教士活动、兴办学校进行文化奴化等。四种方式相互配合构成完整的侵略体系。

#### 3. [来源：1-大工题库（分章节）.md]

资本-帝国主义列强对中国进行文化渗透的方式主要是 ( )

A. 披着宗教外衣，进行侵略活动  B. 为侵略中国制造舆论  C. 控制中国的现代交通运输  D. 投资中国工业
**答案：AB**

**解析：** 列强进行文化渗透的主要方式：一是利用传教士进行殖民主义宣传，许多传教士实际充当了侵略的先遣队（如德国传教士是山东巨野教案的导火索）；二是散布"种族优劣论""黄祸论"等侵略舆论，为侵略制造理论依据。C项"控制现代交通运输"属于经济掠夺，D项"投资中国工业"属于经济侵略，均非文化渗透。
