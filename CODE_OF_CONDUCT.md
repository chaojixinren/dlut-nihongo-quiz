# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放友好的环境，作为贡献者和维护者，我们承诺让参与本项目和社区的每个人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教或性身份。

我们承诺以开放、包容、友善、尊重、健康的方式行事和互动。

## 我们的准则

有助于为我们社区营造积极环境的行为示例包括但不限于：

- 对他人展现同理心和善意
- 尊重不同的主张、观点和感受
- 给予和优雅地接受建设性反馈
- 承担责任并向受我们错误影响的人道歉，并从中吸取经验
- 关注的不仅是我们个人，更是整个社区

不可接受的行为示例包括但不限于：

- 使用性化语言或意象，以及任何形式的性骚扰或挑逗
- 钓鱼、侮辱或贬损的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息，如物理地址或电子邮箱
- 其他可被合理认定为不适当的专业环境行为

## 执行责任

社区领导者负责澄清和执行我们社区的行为标准，并将根据其认为适当的措施，对任何被认为不适当、威胁、冒犯或有害的行为做出回应。

社区领导者有权和有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、issue 和其他贡献，并可在适当时说明制裁的理由。

## 适用范围

本行为准则在所有社区空间生效，并在个人在公共空间正式代表社区时同样适用。

## 执行

辱骂、骚扰或其他不可接受的行为可通过私下联系仓库所有者（在 GitHub profile 上找联系方式）进行举报。所有投诉都会被及时、公正地审阅和处理。

所有社区领导者都有义务尊重举报者的隐私与安全。
